Definición

El tag <em> es reconocido por su capacidad para identificar la enfatización cursiva en el texto.

Ejemplo de esta condición es el siguiente enunciado:

La frase que estoy escribiendo se está escribiendo en cursiva.

Si es que se desea probar por si mismo este tag, le recomendamos este link.

    Lista de atributos para <em>

  • Spellcheck; corrección automática.
  • Lang; define el lenguaje usado.
  • Id; identificador.
  • Draggable; define si puede ser arrastrado.
  • Dir; dirección del texto.
