Distintos properties

Al definir un elemento usamos el atributo style

< elemento style = " property : valor ; " >

Color: style = "color: color ;"

Font: style = "font-family: font ;"

Alineación: style = " text-align : alineación ; "

Tamaño letra: style = " font-size : porciento % ;"

style="color:blue; font-family:'Verdana'; text-align: center; font-size:80%;"
